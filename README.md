# Rubikey

Rubikey is another Ruby library for verifying, decoding, decrypting and parsing YubiKey OTPs.

## Installation

Add this line to your application's Gemfile:

    gem 'rubikey'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rubikey
